final / abstract / static

修飾子対象意味
final classクラス継承不可。
final methodメソッドオーバーライド不可。
final variable変数再代入不可。参照先オブジェクトの状態変更とは別。
abstract classクラスインスタンス化不可。抽象メソッドを持てる。
abstract methodメソッド本体なし。具象サブクラスで実装が必要。
staticフィールド/メソッド/ネストクラスクラス所属。トップレベルクラスには使えない。

sealed

public sealed class A permits B, C {}
final class B extends A {}
non-sealed class C extends A {}

sealedを継承した直接サブクラスは、final、sealed、non-sealed のいずれかが必要です。
