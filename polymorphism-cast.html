ポリモーフィズム・キャスト

宣言型で見える範囲と、実体型で実行されるメソッドを分けます。

宣言型で見えるもの

Object o = "abc";
// o.length(); // NG: Object型にlengthはない
String s = (String) o;
System.out.println(s.length()); // OK

キャストの2段階判定

段階判定例
コンパイル時型階層として変換可能か無関係なfinal型へのキャストは不可になりやすい。
実行時実体がその型として扱えるか扱えなければClassCastException。

instanceofパターンマッチング

Object obj = "java";
if (obj instanceof String s) {
    System.out.println(s.length());
}
// 条件がtrueの範囲でsをStringとして使える。

フィールドはポリモーフィックではない

class A { int x = 1; int get(){ return x; } }
class B extends A { int x = 2; int get(){ return x; } }
A a = new B();
System.out.println(a.x);   // 1
System.out.println(a.get());// 2
