同一性・同値性比較

== は参照や値の同一性、equals はクラスごとの同値性です。

基本

比較primitive参照型
==値を比較同じオブジェクト参照かを比較。Stringなど一部は定数プールに注意。
equals使えないメソッド。String/List/recordなどは内容比較。未オーバーライドならObjectの同一性比較。

Stringの宣言別比較

String a = "java";
String b = "java";
String c = new String("java");
String d = c.intern();

System.out.println(a == b);      // true
System.out.println(a == c);      // false
System.out.println(a == d);      // true
System.out.println(a.equals(c)); // true

コンパイル時定数と実行時連結

String a = "java";
String b = "ja" + "va"; // コンパイル時定数
String x = "ja";
String c = x + "va";    // 実行時に新しいString
final String y = "ja";
String d = y + "va";    // final定数なので定数扱い

System.out.println(a == b); // true
System.out.println(a == c); // false
System.out.println(a == d); // true

配列・List・record

int[] a = {1, 2};
int[] b = {1, 2};
System.out.println(a.equals(b)); // false: 配列はObjectのequals

java.util.List<Integer> l1 = java.util.List.of(1, 2);
java.util.List<Integer> l2 = java.util.List.of(1, 2);
System.out.println(l1.equals(l2)); // true

record Point(int x, int y) {}
System.out.println(new Point(1,2).equals(new Point(1,2))); // true
