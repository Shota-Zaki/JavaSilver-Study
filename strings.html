文字列・比較

String、StringBuilder、==、equals、intern、null比較、メソッド戻り値をまとめます。

Stringは不変

Stringのメソッドは、元の文字列を変更しません。変更されたように見える処理は、新しいStringを返しています。戻り値を受け取らなければ結果は捨てられます。

String s = "abcdef";
s.substring(2);
System.out.println(s); // abcdef

s = s.substring(2);
System.out.println(s); // cdef
試験では replace、substring、toUpperCase などの戻り値未代入がよく出ます。Stringは変わりません。

== と equals

<code>==</code> は同じオブジェクトを参照しているかを見る比較です。Stringの文字内容を比較したいなら <code>equals()</code> を使います。

String a = "java";
String b = "java";
String c = new String("java");

System.out.println(a == b);      // true
System.out.println(a == c);      // false
System.out.println(a.equals(c)); // true

文字列リテラルは文字列プールに置かれます。同じリテラルは同じインスタンスを参照するため、<code>a == b</code> がtrueになります。一方、<code>new String("java")</code> は新しいインスタンスを作るので、参照は別です。

String a = "ja" + "va";  // コンパイル時定数。"java" と同じプール参照
String part = "ja";
String b = part + "va";   // 実行時に作られる別インスタンス
final String fixed = "ja";
String c = fixed + "va";  // final定数なのでコンパイル時定数

System.out.println(a == "java"); // true
System.out.println(b == "java"); // false
System.out.println(c == "java"); // true

intern()

<code>intern()</code> は、文字列プール上の同じ内容のStringへの参照を返します。newで作ったStringでも、intern後はプール側の参照と比較できます。

String a = new String("java");
String b = a.intern();
String c = "java";

System.out.println(a == c); // false
System.out.println(b == c); // true

StringBuilder

StringBuilderは可変です。appendやdeleteなどは同じインスタンスを変更し、そのインスタンス自身を返します。Stringとは逆です。

StringBuilder sb = new StringBuilder("abc");
StringBuilder ret = sb.append("def");

System.out.println(sb);       // abcdef
System.out.println(sb == ret); // true

StringBuilderはequalsを内容比較用にオーバーライドしていません。内容が同じでも、別インスタンスならequalsはfalseです。

StringBuilder a = new StringBuilder("x");
StringBuilder b = new StringBuilder("x");

System.out.println(a == b);      // false
System.out.println(a.equals(b)); // false
System.out.println(a.toString().equals(b.toString())); // true

nullとの比較

nullかもしれない変数でメソッドを呼ぶとNullPointerExceptionになります。安全に比較するなら、リテラル側からequalsを呼ぶ書き方があります。

String s = null;

// s.equals("A");      // NullPointerException
System.out.println("A".equals(s)); // false
System.out.println(s == null);     // true
