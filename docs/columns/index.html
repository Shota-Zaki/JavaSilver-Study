資格選び・受験準備

Java Bronze・Silver・Goldの違い受験順を目的別に整理。Java Bronzeは必要？Bronzeから始める人とSilver直行の違い。申し込み方法・試験時間・問題数まとめ試験番号、時間、合格ライン、受験チケット。Oracle Java試験チケットの買い方楽天チケット、種別、有効期限、購入前確認。再受験無料キャンペーン例年3〜5月のキャンペーン活用と注意点。Gold SE17の受験資格前提資格とSilver後のロードマップ。

参考書・勉強法・勉強時間

参考書まとめBronze、Silver、Goldの教材を1ページで比較。Oracle Java試験の勉強時間Bronze、Silver、Gold別の目安と到達状態。Bronze勉強法未経験者向けの基礎、参考書、演習・模試。Silver勉強法基礎固め、コード読解、単元別演習、固定模試。Gold勉強法API仕様、Gold演習、固定模試、復習導線。未経験からSilver合格まで基礎、参考書、演習、模試の学習計画。Silverの勉強時間現在地別の学習量と復習の考え方。Goldの勉強時間Silver後の学習時間と分野別の重さ。

Silver攻略

Silver SE11とSE17の違い今から受けるならどっちか。Silverで間違えやすい10パターン継承、例外、String、ラムダ式など。Silverに落ちた後にやること再受験までの原因分析と復習順。Java Silverで落ちる理由不合格につながりやすい勉強の偏りを整理。Silver模試の解くタイミング模試後の見直し方。

Silver論点攻略

継承問題の解き方参照型、実体型、override、overload。例外処理問題の解き方checked、catch順序、finally。ラムダ式・関数型インターフェースターゲット型、Predicate、Function。Date and Time API問題LocalDate、Period、Duration、format。コンパイルエラー問題の解き方型、アクセス修飾子、override、例外。出力問題の解き方実行順、評価順、値の変化。

Gold攻略・論点攻略

Goldでつまずきやすい論点Stream、NIO、JDBC、並行処理、モジュール。Stream API問題の解き方中間操作、終端操作、Collectors。入出力・NIO.2問題の解き方Path、Files、try-with-resources。並行処理問題の解き方ExecutorService、Future、synchronized。Gold SE17ロードマップ前提資格と学習順。Gold勉強法参考書、API仕様、演習、模試。
