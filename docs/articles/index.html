基礎・Bronze

実行環境・コンパイル

JDK・JVM・javac・javaコンパイル、実行、クラスパス、package、mainメソッド。 コンパイルエラーの読み方型不一致、名前解決、構文ミス、例外処理漏れ。

型・変数・基本文法

型・変数・リテラルプリミティブ型、参照型、数値リテラル、null、var。 数値リテラル・ラッパークラス・Numberリテラル、ボクシング、実体クラス、Numberの関係。 配列・Stringの基礎配列の作成、添字、Stringの基本操作。 「型」と「実体（インスタンス）」クラス、型、変数、参照、宣言型、実行時型を初歩から整理。

分岐・繰り返し

if・for・whileの追跡条件式、本体、更新式、break、continue。 演算子・制御構造評価順、短絡評価、switch、到達不能コード。

クラスの入口

クラス・インスタンス・staticフィールド、メソッド、コンストラクタ、static。 メソッド・引数・戻り値実引数、仮引数、戻り値、シグネチャ。

Silver

実行・宣言ルール

JDK・JVM・javac・javajavac -d、java -cp、package、複数class、main。 コード読解の基本手順コンパイル可否、出力、例外を順に判定。 コンパイルエラーと実行時例外落ちるタイミングを切り分ける。

型・変数・リテラル

型・変数・リテラルの読み方数値リテラル、型昇格、定数式、var、ラッパークラス。 数値リテラル・ラッパークラス・Number数値表記からプリミティブ値、ボクシング、Number型までを整理。 型昇格と代入変換byte、short、char、int、long、float、double。 nullの意味参照がない状態、NPE、空文字との違い。 型と実体の違いコンパイル時型、実行時型、参照、キャストを詳細に整理。

演算子・制御構造

演算子・制御構造の読み方評価順、短絡評価、三項演算子、for、while、switch。 switch文case、default、break、フォールスルー。 switch式・yield文と式、戻り値、yieldの役割。

String・配列・ArrayList

String・StringBuilder・配列・List不変性、文字列プール、参照、List.of、Arrays.asList。 配列の初期値と参照初期値、添字、参照コピー。 配列の共変性宣言型と実体型のずれ。

クラス・メソッド・アクセス制御

クラス・インスタンス・staticフィールド、コンストラクタ、static、初期化。 メソッド呼び出し解決オーバーロード、型変換、null。 アクセス修飾子public、protected、private、パッケージアクセス。

継承・interface・ポリモーフィズム

継承・interface・ポリモーフィズムsuper、this、is-a、override、default method。 defaultメソッドinterfaceの実装、衝突、継承。 キャストとinstanceof参照型変換、実体型、ClassCastException。 継承の全ルールメソッド、default、アクセス修飾子、abstract、コンストラクタ、例外。

例外処理

例外処理の仕組みと実行順throw、throws、catch、finally、try-with-resources。 実行時例外の原因NPE、配列範囲外、キャスト失敗、parse失敗。

ジェネリクスとワイルドカード型安全性、型消去、ワイルドカード、境界型。 Java API仕様の読み方戻り値、型パラメータ、例外、static / instance。
