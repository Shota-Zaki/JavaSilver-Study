先に結論

Bronzeは必須ではありません。ただし、Javaを初めて学ぶ人には基礎固めとして有効です。Silverの問題で、型・配列・クラス・制御構文の読み方に詰まるなら、Bronze相当の内容から戻った方が早く進みます。

  • クラス、フィールド、メソッドの違いを説明できるか
  • コンパイルエラーと実行時例外を分けられるか
  • Silverの継承問題で参照型と実体型を分けられるか

結論

Java未経験者は、Bronzeを受験するかどうかに関わらず、Bronze相当の基礎から始めます。すでにJavaの基本文法を書いた経験があり、クラス、メソッド、配列、例外の入口が分かる人はSilverから進めてもよいです。

Bronzeから始めるべき人

  • Javaを初めて学ぶ
  • プログラミング経験が少ない
  • 型、変数、if、for、配列で止まりやすい
  • クラスとインスタンスの区別が曖昧
  • 試験に慣れる目的で低い難易度から始めたい

Bronzeは、基礎文法を一通り確認する用途に向きます。

Silverから始めてよい人

  • Javaの基本文法を一通り学んだ
  • 簡単なクラスを作った経験がある
  • コンパイルエラーと実行時例外を分けて考えられる
  • 他言語経験があり、Java文法の差分を学べば進められる

Silver直行の場合でも、苦手分野が出たらBronze範囲へ戻る判断を入れます。

Silver直行のリスク

Silverでは、継承、interface、例外、コレクション、ラムダ式などが出ます。基礎文法が曖昧な状態で進めると、解説を読んでも判断手順が残りにくくなります。

特に、配列、String、クラス、コンストラクタ、staticが弱い場合はBronze相当の演習を挟みます。

当サイトでの判断方法

まずBronze演習を少量解き、型、配列、制御構文、クラス基礎で止まるか確認します。問題文の意味は分かるがSilverの論点で迷う場合は、Silver演習へ進みます。
