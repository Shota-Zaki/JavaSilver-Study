申し込みの流れ

  1. 受験する試験番号を決める
  2. 日本語試験か英語試験かを確認する
  3. 必要に応じて受験チケットを購入する
  4. ピアソンVUEでOracle試験を検索する
  5. 試験番号、受験形式、日時を選ぶ
  6. 支払い画面で受験チケット番号またはプロモーションコードを入力する
  7. 予約完了メールと受験条件を確認する

再受験無料キャンペーンを使う場合は、1回目の予約時点でプロモーションコードを入力します。

申し込み前の注意点

  • 試験番号を間違えると、受けたい資格と違う試験を予約することがあります。
  • 日本語のJava認定資格を狙う場合は、日本語試験の扱いを確認します。
  • 受験チケットには有効期限があります。購入だけ先に行うと期限切れリスクがあります。
  • オンライン受験は本人確認、端末、カメラ、ネットワーク条件を事前に確認します。
  • Goldは前提資格があるため、認定パスを確認します。

受験前にやること

申し込み後は、試験範囲を広げるよりも、模試と復習に寄せます。Silverはコンパイル可否と実行順、GoldはAPI仕様と例外条件を重点確認します。
