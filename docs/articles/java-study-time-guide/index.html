勉強時間

Oracle Java試験の勉強時間はどれくらい？Bronze・Silver・Gold別に解説

Oracle認定Java試験の勉強時間を、Bronze、Silver、Gold別に、未経験者・経験者・再受験者の現在地に分けて整理します。

最終更新日：2026年7月7日。Java SE 17の試験範囲を中心に、演習前後の読み方を整理しています。

先に結論

勉強時間は、受験区分よりも現在地で大きく変わります。完全未経験者は基礎文法に時間を使い、Silver経験者はGoldでAPI仕様の整理に時間を使います。目安時間は、模試で合格ラインを安定して超える状態から逆算します。

見る基準
  • 完全未経験か、他言語経験があるか
  • Javaの基礎構文を読めるか
  • Silver範囲の継承・例外・ラムダを説明できるか
  • Gold範囲のAPI仕様を問題文で判断できるか
  • 週に使える学習時間が固定できるか
Java Practice Labでの使い方勉強時間は、読書時間と演習時間を分けて記録します。点数が伸びない場合、参考書を読む時間ではなく、間違えた問題を読み直す時間が不足しているケースが多いです。

結論：勉強時間は現在地で大きく変わる

同じJava試験でも、未経験者、他言語経験者、Java経験者、再受験者では必要な時間が変わります。時間だけで合格を判断せず、どの問題をどう読めるようになったかで判断します。

試験未経験者の目安Java基礎ありの目安重視すること
Bronze40〜80時間20〜40時間文法・型・制御構文・クラス基礎
Silver100〜150時間50〜90時間コンパイル可否、実行順、継承、例外、ラムダ式
GoldSilver合格後に進める60〜130時間Stream、NIO、並行処理、JDBC、モジュール

Bronzeの勉強時間

Bronzeは、Javaの入口です。未経験者は文法を読む時間と、コードの値を追う時間を分けます。参考書を読むだけではなく、演算子、if、for、配列、クラス基礎を小さい問題で確認します。

  • 最初の10〜20時間：型、変数、演算子、条件分岐、繰り返し。
  • 次の10〜20時間：配列、String、メソッド、クラス基礎。
  • 仕上げ10〜30時間：演習、誤答復習、模試。

Silverの勉強時間

Silverでは、コード読解の練習が中心になります。特に継承、例外、String / StringBuilder、配列、List、ラムダ式は、問題を解きながら判断順を作ります。

既存の詳細記事では、現在地ごとの目安と模試を使うタイミングをさらに細かく整理しています。

Goldの勉強時間

Goldは、Silverの延長ではありますが、API仕様の比重が大きくなります。Stream、Optional、Collectors、NIO.2、並行処理、JDBC、ローカライズ、モジュールは、暗記よりも戻り値・例外条件・実行順の整理します。

時間より重要な到達状態

勉強時間を増やしても、次の状態になっていない場合は合格点に届きにくいです。

  • コンパイルエラーと実行時例外を分けられる。
  • 参照型と実体型を分けてメソッド呼び出しを追える。
  • 選択肢を見る前にコードの状態を整理できる。
  • 正答だけでなく、誤答選択肢を除外できる。
  • 模試の結果から弱点単元へ戻れる。

学習計画への落とし込み

平日30分中心なら、短い演習と復習を毎日行い、休日に模試を入れます。平日に参考書だけを読むと判断練習が不足します。
