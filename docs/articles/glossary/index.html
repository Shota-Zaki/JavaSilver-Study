Java最初に使う用語

用語だけを別の用語で言い換えず、短いコードと結び付けて説明します。詳細な規則は各単元の記事へ進んでください。

広告スポンサーリンク表示エリア

値・変数・型

int count = 3;
String name = "Java";
値
3や"Java"のように、プログラムが扱うデータです。
変数
countやnameのように、値または参照を保持する名前です。
型
intやStringのように、保持できる値と可能な操作を決める分類です。
宣言
int countの部分です。変数名と型をプログラムへ知らせます。
初期化
変数を宣言すると同時に最初の値を代入することです。上の= 3、= "Java"が該当します。

式・評価・文

int total = count + 2;
System.out.println(total);
式
count + 2のように、計算すると値と型が決まるコードです。
評価
式を実際に計算して結果を得ることです。countが3なら、count + 2の評価結果は5です。
文
int total = count + 2;のような処理の単位です。多くの文はセミコロンで終わります。
戻り値
メソッドが呼び出し元へ返す値です。name.length()は文字数をintで返します。

クラス・オブジェクト・インスタンス・参照

class User {
    String name;
}

User user = new User();
user.name = "A";
クラス
Userのように、作成するオブジェクトのフィールドやメソッドを定義する型です。
オブジェクト
new User()で作られる実体です。
インスタンス
特定のクラスから作られたオブジェクトを指します。上のオブジェクトはUserクラスのインスタンスです。
参照
オブジェクトの場所を指すための値です。変数userはUserオブジェクトそのものではなく、そのオブジェクトへの参照を保持します。
null
参照型変数がどのオブジェクトも参照していないことを表す値です。

フィールド・メソッド・引数

class Calculator {
    int lastResult;                  // フィールド

    int add(int left, int right) {   // leftとrightは仮引数
        lastResult = left + right;
        return lastResult;
    }
}

int answer = new Calculator().add(2, 3); // 2と3は実引数
フィールド
オブジェクトまたはクラスが保持する変数です。上ではlastResultです。
メソッド
クラス内に定義した処理です。上ではaddです。
仮引数
メソッド宣言側で受け取る変数です。leftとrightが該当します。
実引数
メソッド呼び出し側から渡す式です。2と3が該当します。

コンパイルと実行

ソースコード
.javaファイルに書いたJavaのプログラムです。
コンパイル
javacがソースコードを検査し、通常は.classファイルへ変換する処理です。構文や型が不正なら、ここでエラーになります。
実行
JVMがクラスを読み込み、mainメソッドなどの命令を処理する段階です。
実行時例外
コンパイルは通ったコードが、実行中の値や状態により続行できなくなったときに投げられる例外です。

コンパイルエラー、実行時例外、正常終了は別の段階です。コードではこの順番で判定します。

記事末尾の確認

  • 値、変数、型を一つの例で区別できる。
  • 式を評価すると値と型が決まると説明できる。
  • 参照変数とオブジェクトを同じものとして扱わない。
  • 仮引数と実引数を呼び出し側・宣言側で区別できる。
