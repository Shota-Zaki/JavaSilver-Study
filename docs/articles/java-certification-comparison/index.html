Java Bronze・Silver・Goldの違い｜どれから受けるべきか

Bronze、Silver、Goldは、同じJava資格でも対象者と問われ方が違います。最初に受ける試験を誤ると、参考書や演習の選び方もずれます。

広告スポンサーリンク表示エリア

最終更新日：2026年7月7日。試験制度や認定条件は変更されるため、受験前に公式情報を確認してください。

先に結論

受験順は、現在のJava理解度で決めます。完全未経験ならBronze相当の基礎、基本文法を一通り読めるならSilver、Silver取得済みまたは同等の読解力があるならGoldです。資格名だけで難易度を選ぶと、参考書を読んでも演習で詰まりやすくなります。

見る基準
  • Javaの変数・制御構文・配列を読めるか
  • 継承・interface・例外処理をコードで追えるか
  • Stream、入出力、並行処理などのAPI仕様まで必要か
  • 受験目的が基礎固め、就職準備、実務寄りの整理のどれか
Java Practice Labでの使い方迷う場合は、単元別解説で基礎を確認してから通常演習を数問解きます。問題文のコードを追えない段階ならBronze相当、誤答理由まで読めるならSilver、API仕様の穴を潰したいならGoldに進みます。

結論

Java未経験者はBronze相当の基礎から始めます。Javaの基本文法を一通り読める人はSilver SE 17、Silver取得済みまたは実務でJavaを使っている人はGold SE 17を検討します。

区分対象者主な目的当サイトの入口
Bronze未経験者・学習初期Java文法とオブジェクト指向の入口を確認するBronze演習
SilverJava初級者基本文法をコード読解で使える状態にするSilver演習
GoldSilver取得者・中級者API仕様と実装判断を広く確認するGold演習

Bronzeは基礎の確認用

Bronzeは、Java言語未経験者向けの入門資格です。型、変数、演算子、分岐、繰り返し、配列、クラス、継承の入口を確認します。

受験が必須ではない人もいますが、Silverの問題で用語や基本文法で止まる場合は、Bronze範囲へ戻った方が時間を使いやすい進め方です。

SilverはJava初級者の中心資格

Silverは、Javaアプリケーション開発に必要な基本的なプログラミング知識を問う資格です。Bronzeよりも、コードの実行順、参照型と実体型、例外、コレクション、ラムダ式などの判断が重要になります。

新しく受験するなら、基本的にはSilver SE 17を主軸にします。古い教材を使う場合は、SE11とSE17の差分を確認します。

GoldはSilverより上のAPI・設計判断

Goldは、Stream、Optional、NIO、JDBC、並行処理、ローカライズ、モジュールなどを扱います。文法知識だけでなく、API仕様を読み分ける力を使います。

Gold SE 17は前提資格があるため、受験前に認定パスを確認します。Silver範囲が曖昧なら、先にSilverの復習を優先します。
