Gold SE 17の前提資格

Gold SE 17を取得するには、Java SE 17 Programmer II（1Z0-826-JPN）への合格に加え、認定パス上の前提資格を満たします。Silver SE 17、Silver SE 11、Silver SE 8、Silver SE 7などが前提資格として扱われます。

受験そのものと認定取得の条件は分けて確認します。合格しても前提資格が不足すると、認定取得の扱いが想定とずれることがあります。

先に潰すべき弱点

Goldで詰まりやすい人は、Silver範囲の型、継承、ラムダ式、例外処理が曖昧なまま進んでいることがあります。Gold問題に入る前に、次のページで基礎を確認します。

  • 継承・interface・override
  • 例外処理完全整理
  • ラムダ式と関数型インタフェース
  • Stream API完全整理

当サイトの使い方

Gold演習では、問題ごとにコンパイル可否、戻り値、例外条件、評価順を確認します。模試後は正答率だけでなく、タグ別・単元別に弱点を見ます。
