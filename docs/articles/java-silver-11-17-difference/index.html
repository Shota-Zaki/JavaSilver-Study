先に結論

今から新しく受験するなら、通常はSE17を主軸に考えます。ただし、手元にSE11教材がある場合でも全部捨てる必要はありません。共通範囲を使い、SE17差分を追加で確認します。

結論

今から新しく受験するなら、原則としてSilver SE 17を主軸にします。Silver SE 11教材を持っている場合でも、基本文法、クラス、継承、例外、コレクションなど共通範囲の学習には使います。ただし、SE17で追加・強調される論点は別途確認します。

学習上の違い

Silver SE17では、テキストブロック、switch式、record、sealed、instanceofパターンなど、Java 17世代の構文が学習対象に入ります。単に暗記するのではなく、コンパイル可否と実行時の挙動を分けて読みます。

  • recordの定義
  • sealedとpermits
  • switch式とyield
  • instanceofパターン

教材の選び方

新しく買うならSE17対応の参考書・問題集を選びます。SE11教材をすでに持っている場合は、共通範囲の復習に使い、SE17固有の論点は当サイトの単元別解説やSE17対応問題で補います。
