Bronze直前整理

Bronze範囲を短時間で見直すためのページです。深掘りは基礎解説、直前整理はこのページで行います。

広告スポンサーリンク表示エリア

最初に見ること

  • Javaプログラムはclassを単位に書く
  • 実行入口はmainメソッド
  • public class名とファイル名は一致させる
  • package宣言はimportより前に書く

文法の見直し

  • 変数、型、リテラルを分ける
  • if / switch は条件と分岐先を読む
  • for / while / do-while は条件判定と更新の順番を追う
  • 配列はlength、Stringはlength()を使う

オブジェクト指向の入口

  • クラスは設計図、インスタンスは実体
  • フィールドはインスタンスが持つ値
  • メソッドは処理
  • コンストラクタはインスタンス生成時の初期化処理
  • staticはインスタンスなしでクラス側に属する

直前に見直す注意点

  • mainの引数や戻り値を崩す
  • ローカル変数を初期化せずに使う
  • Stringの比較で==とequalsを混同する
  • 配列の範囲外アクセスを見落とす
  • void Sample() をコンストラクタとして読んでしまう
