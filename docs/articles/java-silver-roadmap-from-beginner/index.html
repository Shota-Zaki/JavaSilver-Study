Java未経験からOracle Java Silverに合格するまでの学習計画

未経験からSilverを目指す場合、いきなり問題集だけを解くより、基礎文法、参考書、演習、模試の順に進める方が安定します。

広告スポンサーリンク表示エリア

最終更新日：2026年7月7日。Java SE 17の試験範囲を中心に、演習前後の読み方を整理しています。

先に結論

未経験からSilverを狙う場合、最短でも基礎文法、Silver範囲、問題演習、模試復習の4段階に分けます。いきなり問題集を回すより、読めないコードを減らしてから模試へ進む方が誤答理由を特定できます。

見る基準
  • 基礎文法に戻る期間を確保しているか
  • Silver参考書を読む期間と演習期間を分けているか
  • 模試を試験直前だけに残していないか
  • 不正解の理由を単元別に記録しているか
Java Practice Labでの使い方ロードマップは固定ではありません。演習で継承が弱ければ継承へ戻り、例外が弱ければ例外へ戻す形で、模試結果から逆算します。

全体の流れ

  1. Bronze相当の基礎文法を学ぶ
  2. Silver SE17対応の参考書を1周する
  3. 当サイトの単元別演習で弱点を確認する
  4. 問題集を2〜3周する
  5. 固定模試で時間配分を確認する
  6. 結果画面から弱点単元へ戻る

1か月目：基礎文法

変数、型、if、switch、for、配列、String、クラス、メソッド、コンストラクタを優先します。ここで詰まるとSilverの継承や例外で止まりやすくなります。

2か月目：Silver範囲

継承、interface、override、overload、例外、コレクション、ラムダ式、Date and Time APIを参考書で確認します。章ごとに単元別演習を解き、間違えた分野を記録します。

3か月目：模試と復習

模試では時間配分を確認します。点数だけで判断せず、迷った問題、正答したが根拠が曖昧な問題、コンパイル可否を誤った問題を復習対象にします。

Silver模試の使い方へ

短期で進める場合

1〜2か月で進める場合は、参考書を完璧に読むより、章ごとに演習へ移る方がよいです。ただし、基礎文法が曖昧なまま模試を連発すると、復習効率が落ちます。
