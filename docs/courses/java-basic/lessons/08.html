Java基礎 Lesson 8

if文と条件式

if文は、trueかfalseになる条件で処理を分けます。比較演算子と短絡評価を順に読みます。

ここまでできればOK

条件式が成立するか、どの処理が実行されるかを追えるようになります。

まず押さえる言葉

Javaのif文は数値を真偽値として扱いません。丸かっこ内の式は必ず`boolean`へ評価される必要があります。

条件式とはtrue または false になる式。
booleanとは真偽値だけを扱う型。true/false。
比較演算子とは>、==など、比較結果をbooleanで返す演算子。
短絡評価とは&&や||で右側の評価が省略されること。
ブロックとは{ }で囲んだ処理のまとまり。

if文と条件式の決まり

項目意味・使い方具体例
条件式評価結果がbooleanになる式。age >= 18
if文条件がtrueのときに文またはブロックを実行します。if (condition) { ... }
else直前の対応可能なifに結び付きます。波かっこで対応関係を明示できます。
比較演算子<、<=、>、>=、==、!=。結果型はbooleanです。
短絡AND&&は左辺がfalseなら右辺を評価しません。危険な処理を右辺で避けられます。
短絡OR||は左辺がtrueなら右辺を評価しません。右辺の副作用が実行されない場合があります。

ifの基本

if (条件式) {
    条件がtrueのときの処理
} else {
    条件がfalseのときの処理
}

条件式に書けるのはtrueまたはfalseになる式。if (1)やif (0)はJavaでは使えない。

比較演算子

演算子意味例
==等しいa == b
!=等しくないa != b
<小さいa < b
>大きいa > b
<=以下a <= b
>=以上a >= b
参照型の==

Stringなどの参照型で==を使うと、内容ではなく参照先が同じかを見る。

短絡評価

&&は左がfalseなら全体がfalseなので右を評価しない。||は左がtrueなら全体がtrueなので右を評価しない。

右側が実行されない例

int x = 0;
if (x > 0 && ++x > 0) {
    System.out.println("A");
}
System.out.println(x); // 0

&や|をbooleanに使うと短絡しない。右側も評価される。

代入と比較の混同

これはコンパイルできる

boolean flag = false;
if (flag = true) {
    System.out.println("run");
}

flag = trueは代入式で、結果としてtrueになるためifに書ける。比較のつもりならflag == trueだが、通常はif (flag)で十分。

条件式と短絡評価の最小例

`Main.java`として保存する完全なコード

public class Main {
    public static void main(String[] args) {
        String text = null;
        if (text != null && text.length() > 0) {
            System.out.println(text);
        } else {
            System.out.println("empty");
        }
    }
}

左辺text != nullがfalseなので、右辺text.length() > 0は評価されません。そのため、このコードではNullPointerExceptionは発生しません。

似た言葉との違い

混同しやすいもの違い見分け方
= / ===は代入、==は比較。条件式内で代入していないか見る。
&& / &&&は短絡評価あり。&は右辺も評価する。右辺に副作用があると差が出る。
数値 / booleanJavaでは数値を真偽値として使えない。if (x)はxがbooleanでなければ不可。

条件分岐の確認順

  1. 条件式全体がboolean型か確認する。
  2. 演算子の優先順位に従って式をグループ化する。
  3. オペランドを左から評価する。
  4. &&または||なら短絡条件を適用する。
  5. 最終結果がtrueの分岐だけを実行する。

確認用コード

int a = 10;
int b = 0;
if (a < 5 && ++b > 0) {
    a++;
}
System.out.println(a + ":" + b);
見る点このコードでの判断結論
左辺a < 5はfalse&&なので右辺を評価しない
++b実行されないbは0のまま
出力aは10、bは010:0

ここまでの要点

  • ifの条件式がbooleanになるか確認できる。
  • =と==を区別できる。
  • &&と||の短絡評価で右辺が実行されるか判断できる。
