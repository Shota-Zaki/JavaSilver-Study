ミニ演習

短いコードで、コンパイル可否・実行時例外・出力を即判定する練習ページです。

型変換

byte b = 10; b = b + 1;

コンパイルエラー。b + 1 は int。

byte b = 10; b += 1;

OK。複合代入は暗黙キャストを含みます。

float f = 1.0;

コンパイルエラー。1.0 は double。

String

String s = "A"; s.concat("B"); System.out.println(s);

A。Stringは不変で、戻り値を受け取っていません。

String s = null; System.out.println(s);

null と表示。メソッド呼び出しではありません。

String s = null; System.out.println(s.length());

NullPointerException。

配列・List

int[] a = new int[0]; System.out.println(a.length);

0。要素数0の配列は作れます。

int[] a = new int[0]; System.out.println(a[0]);

ArrayIndexOutOfBoundsException。

List.of("A").add("B");

コンパイルは可能ですが、実行時に UnsupportedOperationException。

継承

Parent p = new Child(); p.childOnly();

Parent に childOnly がなければコンパイルエラー。宣言型で判定します。

interfaceの void work(); を void work(){} で実装

コンパイルエラー。実装側は public が必要です。

sealedクラスの許可サブクラスに修飾子なし

コンパイルエラー。final/sealed/non-sealed のいずれかが必要です。

例外

catch(Exception) の後に catch(IOException)

コンパイルエラー。後者が到達不能です。

try-with-resourcesで A a; B b; のclose順

b.close() → a.close()。宣言の逆順です。

Integer.parseInt("12a")

NumberFormatException。
