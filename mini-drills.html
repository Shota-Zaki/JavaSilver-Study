ミニ演習

短いコードで、間違えやすい判定だけ練習します。

1. String比較

String a = "ja" + "va";
String x = "ja";
String b = x + "va";
System.out.println(a == "java");
System.out.println(b == "java");

答え: true / false。変数を含む連結は実行時に新しいStringになりやすい。

2. byte演算

byte b = 10;
// b = b + 1;
b += 1;
System.out.println(b);

答え: 11。代入文は不可だが、複合代入は暗黙キャストを含む。

3. do-while

int i = 0;
do {
    System.out.print(i);
} while (++i < 3);
System.out.println("/" + i);

答え: 012/3。前置なので条件判定前に増える。

4. override

class A { void test(){ System.out.println("A"); } }
class B extends A { void test(){ System.out.println("B"); } }
A a = new B();
a.test();

答え: B。overrideメソッドは実体型で実行。

5. finally

try {
    System.out.print("A");
} finally {
    System.out.print("B");
}

答え: AB。例外がなくてもfinallyは実行される。
