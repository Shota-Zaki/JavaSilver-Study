ミニ演習

1. 後置++

int i = 0;
System.out.println(i++ + i);

答え: 1。左のi++は0を使い、その後iが1になる。

2. 複合代入

byte b = 1;
b += 1;
b = b + 1;

3行目がコンパイルエラー。b + 1はint。

3. String比較

String a = "ja";
String b = a + "va";
System.out.println(b == "java");

false。変数を含む連結は実行時生成。

4. override

A a = new B();
a.test();
a.onlyB();

testはBが動くが、onlyBは宣言型Aにないのでコンパイルエラー。

5. finally

try { return; } finally { System.out.println("F"); }

return前にfinallyが実行される。
