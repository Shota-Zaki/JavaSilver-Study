ミニ演習

学習記事で読んだルールを短いコードで確認します。答えは開閉式です。

使い方

先にコードだけ読んで、コンパイル可否・実行時例外・出力を決めてから答えを開いてください。

1. Stringの比較

String a = "ja" + "va";
String b = "java";
System.out.println(a == b);
答え

true。定数式なのでコンパイル時に "java" になり、文字列プールの同じ参照。

2. 変数を含む連結

String x = "ja";
String a = x + "va";
String b = "java";
System.out.println(a == b);
答え

false。実行時連結で新しいStringが作られる。

3. byteの演算

byte a = 10;
byte b = 20;
byte c = a + b;
答え

コンパイルエラー。a + b は int。

4. 複合代入

byte a = 10;
a += 20;
System.out.println(a);
答え

30。複合代入は暗黙キャストを含む。

5. static

class A {
    int value = 10;
    static void test(){ System.out.println(value); }
}
答え

コンパイルエラー。staticメソッドからインスタンスフィールドへ直接アクセスできない。

6. finally

try {
    System.out.print("A");
} finally {
    System.out.print("B");
}
答え

AB。例外がなくてもfinallyは実行される。
