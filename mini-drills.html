ミニ演習

Stringの戻り値

String s = "abc";
s.toUpperCase();
System.out.println(s);
// 答え: abc

Stringは不変です。戻り値を受け取っていないため変化しません。

do-while

int i = 0;
do {
    System.out.print(i++);
} while (i < 0);
System.out.println(i);
// 答え: 01

do-whileは条件判定前に本体を1回実行します。

record比較

record P(String name) {}
P a = new P("A");
P b = new P("A");
System.out.println(a == b);
System.out.println(a.equals(b));
// false / true

==は参照比較、recordのequalsは値比較です。
