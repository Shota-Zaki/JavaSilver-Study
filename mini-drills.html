ミニ演習

短いコードで、コンパイル可否・実行時例外・出力を即判定する練習です。

短問セット

1. byte b=1; b=b+1;

コンパイルエラー。b+1 はintです。

2. byte b=1; b+=1;

OK。複合代入は暗黙キャストを含みます。

3. String s=null; System.out.println(s);

null と表示。メソッド呼び出しではありません。

4. String s=null; s.length();

NullPointerException。

5. Object o="x"; Integer i=(Integer)o;

実行時に ClassCastException。

6. var x=null;

コンパイルエラー。型推論できません。

使い方

各問題を20秒以内に「コンパイルエラー / 実行時例外 / 出力」のどれかへ分類してください。遅い場合は、知識不足ではなく判定順が固定されていない可能性が高いです。
