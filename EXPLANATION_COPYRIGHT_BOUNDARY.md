# 解説強化方針

この学習ページでは、PDF解説をそのまま全文転載しない。

## 採用する形式

- PDF解説の要点
  - PDFの解説内容を、問題固有の判定に必要な範囲で整理する。
  - 長い背景説明や本文の連続転載は避ける。
- 補足解説
  - 本サイト用に新規作成する。
  - Java Silverで間違えやすい判定順、選択肢の切り方、試験上の注意点を追加する。

## 禁止する形式

- PDF本文・解説本文の全文コピー
- 全問に同じ定型文を貼り付けること
- 問題と関係の薄い一般論を水増しすること

## 今回の進捗

- 第1章9問を「PDF解説の要点 + 補足解説」の形式へ更新。
- 第2章以降は既存の要約ベース解説を維持。以後、章ごとにPDF照合しながら更新する。
