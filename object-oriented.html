クラス・static・record

インスタンス、参照、static、final、record、GCを、出力結果とコンパイル判定に使える形で整理します。

クラスとインスタンス

クラスは型の定義で、インスタンスは new によって作られる実体です。参照変数にはインスタンスそのものではなく、インスタンスへの参照が入ります。

class Item {
    int price;
}

Item a = new Item();
Item b = a;
b.price = 100;
System.out.println(a.price); // 100

b = a; はオブジェクトのコピーではありません。同じインスタンスへの参照をコピーしているだけです。

フィールドとローカル変数

同名のローカル変数や引数がある場合、そちらが優先されます。フィールドに明示的にアクセスするなら this を使います。

class Item {
    private int num = 10;

    public void setNum(int num) {
        this.num = num;
    }

    public int getNum() {
        return this.num;
    }
}

インスタンスごとにフィールドの値は別です。別インスタンスの値を書き換えても、他のインスタンスには影響しません。

static

static メンバはインスタンスではなくクラスに属します。staticフィールドは全インスタンスで共有され、staticメソッドはインスタンスなしで呼び出せます。

class Counter {
    static int count;
    int value;

    static void printCount() {
        System.out.println(count);
        // System.out.println(value); // NG。インスタンスメンバへ直接アクセス不可
    }
}
staticメソッドからインスタンスフィールドへ直接アクセスできません。アクセスしたいなら、インスタンス参照を用意する必要があります。

final

final は「再代入できない」「オーバーライドできない」「継承できない」など、付ける対象によって意味が変わります。

final変数

初期化後に再代入できません。参照型なら参照の付け替え不可であり、参照先オブジェクトの状態変更まで禁止するわけではありません。

finalメソッド

サブクラスでオーバーライドできません。

finalクラス

継承できません。

finalフィールド

コンストラクタ終了までに確実に初期化する必要があります。

record

recordは値の組を表すための簡潔なクラス宣言です。コンポーネントから、フィールド、正規コンストラクタ、アクセサ、equals、hashCode、toString が生成されます。

public record Person(String name, int age) { }

Person p = new Person("A", 20);
System.out.println(p.name()); // A
System.out.println(p.age());  // 20
System.out.println(p);        // Person[name=A, age=20]

アクセサ

name() のように、コンポーネント名そのものです。getName() ではありません。

継承

recordは暗黙的にfinalで、他クラスをextendsできません。ただしinterfaceはimplementsできます。

状態

追加のインスタンスフィールドは基本不可。recordヘッダーのコンポーネントが状態です。

コンパクトコンストラクタ

入力値の検証や正規化に使えます。フィールド代入を自分で書く必要はありません。

record User(String name, int age) {
    public User {
        if (age < 0) {
            throw new IllegalArgumentException();
        }
    }
}

ガベージコレクション

どこからも参照されなくなったインスタンスは、ガベージコレクションの対象になります。ただし、対象になることと、即座に回収されることは別です。

Object a = new Object();
Object b = new Object();
Object c = a;
a = null;
b = null;

// bが参照していたインスタンスは対象
// aが参照していたインスタンスはcが参照しているため対象外
