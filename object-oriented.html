クラス・static・record

インスタンス、参照、static、final、ガベージコレクション、recordを整理します。

クラスとインスタンス

クラスは設計図、インスタンスは実体です。変数が直接インスタンスを持つのではなく、参照を持ちます。

Item a = new Item();
Item b = new Item();
System.out.println(a == b); // false: 別インスタンス
b = a;
System.out.println(a == b); // true: 同じ参照

インスタンスフィールドとstaticフィールド

インスタンスフィールドは各インスタンスごとに持ちます。staticフィールドはクラスに1つです。

class Counter {
    int instanceCount;
    static int classCount;
}

Counter a = new Counter();
Counter b = new Counter();
a.instanceCount++;
a.classCount++;
System.out.println(b.instanceCount); // 0
System.out.println(b.classCount);    // 1
staticメンバはインスタンス経由でも呼べてしまう場合がありますが、試験では「共有される」という性質を見ます。

staticメソッドから見えるもの

staticメソッドから、インスタンスフィールドやインスタンスメソッドへ直接アクセスできません。インスタンスが必要です。

class Sample {
    int value;
    static void test() {
        // System.out.println(value); // NG
        Sample s = new Sample();
        System.out.println(s.value);  // OK
    }
}

finalの意味

final は対象によって意味が変わります。変数なら再代入不可、メソッドならoverride不可、クラスなら継承不可です。

対象意味
変数再代入できない
メソッドオーバーライドできない
クラス継承できない
final StringBuilder sb = new StringBuilder("A");
sb.append("B");       // OK: 中身の変更
// sb = new StringBuilder(); // NG: 再代入

ガベージコレクション

どの変数からも参照されなくなったインスタンスは、ガベージコレクションの対象になります。ただし、実際に回収されるタイミングは保証されません。

Object a = new Object();
Object b = new Object();
Object c = a;
a = null;
b = null;
// ここで対象になるのは、もともとbが参照していたインスタンスだけ
// aの元インスタンスはcが参照している

recordの基本

recordはデータを保持するクラスを簡潔に定義する仕組みです。コンポーネント、アクセサ、equals、hashCode、toStringなどが自動生成されます。

record User(String name, int age) {}

User u = new User("A", 20);
System.out.println(u.name()); // A
// u.name = "B";             // NG

ミニ演習

Q. staticメソッドからthisは使える？

使えません。staticメソッドは特定のインスタンスに属さないためです。

Q. final参照型変数の中身は絶対に変更不可？

違います。再代入不可です。参照先オブジェクトが可変なら中身は変更できます。

参照のコピーとオブジェクトのコピーは違う

a = b は、bが見ているインスタンスへの参照をaへコピーするだけです。オブジェクトが複製されるわけではありません。

Item a = new Item();
Item b = new Item();
a = b; // bの参照をaへコピー。もともとのaのインスタンスは参照されなくなる可能性

static初期化の見方

staticフィールドはインスタンスごとではなくクラスごとに1つです。複数インスタンスから更新しても同じ値を共有します。

class Sample {
    static int count;
    Sample() { count++; }
}
new Sample();
new Sample();
System.out.println(Sample.count); // 2
