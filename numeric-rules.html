数値・リテラル・型昇格

byte/short/charのint昇格、複合代入、定数式、アンダースコアをまとめます。

byte/short/charは演算でintになりやすい

byte b = 10;
// byte x = b + 1; // NG: b + 1 はint
byte y = 10 + 1;   // OK: 定数式で範囲内
char c = 'A';
int n = c + 1;     // OK: charは演算時にintへ昇格

複合代入は暗黙キャストを含む

byte b = 10;
b += 1;      // OK: b = (byte)(b + 1) 相当
// b = b + 1; // NG: b + 1 はint
