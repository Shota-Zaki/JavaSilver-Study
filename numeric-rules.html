数値・リテラル・型昇格

byte/short/char、定数式、数値昇格、複合代入、アンダースコアの置き方を、コンパイル判定用に整理します。

数値問題の判定フロー

  1. リテラルの型を決める。整数は基本 int、小数は基本 double。
  2. 代入先の型に暗黙変換できるかを見る。
  3. 式に変数が含まれるなら、byte/short/char は基本 int に昇格すると考える。
  4. += などの複合代入だけは暗黙キャストが入る点を別扱いする。
数値問題は「値が入るか」ではなく「式の型が代入先に合うか」です。ここを混同すると落ちます。

定数式なら狭い型へ代入できる

byte や short へ int リテラルを代入する場合、値が範囲内ならコンパイルできます。定数式も同じです。

byte a = 127;         // OK
        // byte b = 128;      // NG
        short c = 128 + 128;  // OK。定数式で256

        byte x = 10;
        byte y = 20;
        // byte z = x + y;    // NG。変数を含む演算結果はint

見た目の値が小さくても、式の型が int なら byte にそのまま代入できません。

二項数値昇格

算術演算では、byte、short、char は int に昇格します。片方が long なら結果は long、片方が double なら結果は double です。

式結果の型注意
byte + byteintbyte ではない
short + intint小さい方が昇格
int + longlongint へ戻すにはキャスト
float + doubledoublefloat に代入不可
short s1 = 10;
        short s2 = 20;
        // short s3 = s1 + s2; // NG。結果はint
        short s4 = (short)(s1 + s2); // OK

複合代入は暗黙キャストを含む

+=、-=、*=、/= などの複合代入は、通常の代入と違って暗黙のキャストを含みます。

byte b = 10;
        // b = b + 1; // NG。b + 1 はint
        b += 1;       // OK。b = (byte)(b + 1) 相当

ただし、値があふれれば桁あふれした値になります。複合代入は「安全」という意味ではありません。

byte b = 127;
        b += 1;
        System.out.println(b); // -128

charの扱い

char は文字を表しますが、内部的には16ビットの数値として扱われる場面があります。算術演算に入ると int になります。

char c = 'A';      // OK
        int n = c;         // OK。65
        // char d = c + 1; // NG。結果はint
        char e = (char)(c + 1); // OK

String の "A" と char の 'A' は別物です。単一引用符は1文字だけです。

数値リテラルの表記とアンダースコア

0b は2進数、先頭 0 は8進数、0x は16進数です。8進数に 8 や 9 は使えません。

int a = 0b1010; // 10
        int b = 012;    // 10
        int c = 0xA;    // 10
        // int d = 089; // NG

アンダースコアは数字の間にだけ置く、と考えると安全です。

int ok = 1_000_000;
        int alsoOk = 0b1010_0101;
        // int ng1 = _100;
        // int ng2 = 100_;
        // int ng3 = 0x_FF;
        // double ng4 = 3_.14;

ミニ判定

Q1. short s = 128 + 128; は？

OKです。定数式で値が short の範囲内です。

Q2. int i = 2 * 3L; は？

コンパイルエラーです。3L により式全体が long になり、int に暗黙代入できません。

Q3. float f = 10.0; は？

コンパイルエラーです。10.0 は double リテラルです。10.0F にします。
