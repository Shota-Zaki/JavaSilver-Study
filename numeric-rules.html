数値・リテラル・型昇格

整数リテラル、浮動小数点、アンダースコア、byte/short/charの昇格をまとめます。

デフォルト型

リテラルデフォルト型例
整数int10
longlong10L
小数double1.0
floatfloat1.0F
文字char'A'

byte/short/charの昇格

byte a = 10;
byte b = 20;
// byte c = a + b; // コンパイルエラー
int c = a + b;     // OK

byte、short、char は算術演算時に基本的に int へ昇格します。

定数なら代入できる場合

byte a = 10 + 20;      // OK。コンパイル時に30と分かる
byte x = 10;
byte y = 20;
// byte z = x + y;     // NG。演算結果はint

複合代入は暗黙キャストを含む

byte b = 10;
b += 1;       // OK。b = (byte)(b + 1) 相当
// b = b + 1; // NG。b + 1 はint
