ひっかけ集

Java Silverで落とされやすい誤認ポイントを、問題形式の裏側から整理します。

ひっかけの本質

Java Silverのひっかけは、難しいアルゴリズムではなく、仕様の境界を曖昧に覚えている箇所を突いてきます。自分の感覚ではなく、コンパイラと実行時のルールで判断します。

頻出ひっかけ

コンパイル前提崩し

出力を追わせておいて、実は型不一致・アクセス違反・未初期化でコンパイルエラー。

戻り値未使用

Stringメソッドやreplace結果を受け取っていない。

参照コピー

代入でオブジェクトがコピーされたと誤認させる。

短絡評価

右辺の++が実行されるかどうか。

catch順

親例外を先に書いて子catchが到達不能。

override風overload

引数が違うためオーバーライドではない。

解くときの型

選択肢を見る前に、自分で「コンパイル可否」「実行時例外」「出力」を確定させます。選択肢を先に見ると、もっともらしい誤答に引っ張られます。
