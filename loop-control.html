ループ制御

while、do-while、for、break、continue、ラベル付き制御を、出力順の追跡に絞って整理します。

ループの基本判定

ループ問題は、初期化、条件判定、本文、更新式の順に追います。while は最初に条件を見ますが、do-while は本文を1回実行してから条件を見ます。

int n = 0;
do {
    System.out.println(n);
    n++;
} while (n < 0);
// 0 は1回出力される

breakとcontinue

break は現在のループを終了します。continue は現在の反復を終了し、次の反復へ進みます。for文ではcontinue後に更新式が実行される点も忘れないこと。

for (int i = 0; i < 5; i++) {
    if (i == 2) continue;
    if (i == 4) break;
    System.out.print(i);
}
// 013

ラベル付き制御

ネストしたループでは、ラベル付き break / continue で外側のループを対象にできます。試験では、どのループが終了・継続されるかを問われます。

outer:
for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
        if (j == 1) break outer;
        System.out.println(i + ":" + j);
    }
}
// 0:0 のみ
ラベル付き continue outer; は、外側ループの次の反復へ進みます。外側ループ自体を完全に抜けるのは break outer; です。
