演算子・制御構造

前置/後置、短絡評価、switch、ループ制御を例題で確認します。

前置++と後置++

int a = 10;
int b = a++ + ++a;
// a++ は10を使ってから a=11
// ++a は先に a=12 にして12を使う
System.out.println(b); // 22
System.out.println(a); // 12

値を使うタイミングと変数が更新されるタイミングを分けます。

短絡評価

int a = 10;
if (a < 0 && ++a > 0) { }
System.out.println(a); // 10

if (a < 0 & ++a > 0) { }
System.out.println(a); // 11

&&は左辺がfalseなら右辺を評価しません。&はbooleanでも右辺を評価します。

複合代入

byte b = 10;
b += 1;      // OK。暗黙キャストを含む
// b = b + 1; // コンパイルエラー。b + 1 はint

複合代入は単純な書き換えではなく、代入先型への暗黙変換が入ります。

switch文とswitch式

int n = 2;
switch (n) {
  case 1: System.out.println("A");
  case 2: System.out.println("B");
  default: System.out.println("C");
}
// B C

String result = switch (n) {
  case 1 -> "A";
  case 2 -> {
    System.out.println("two");
    yield "B";
  }
  default -> "C";
};
System.out.println(result);
// two
// B

switch文はbreakがないとフォールスルーします。switch式は値を返すため、式全体の末尾にセミコロンが必要です。

  • yield はswitch式全体の返り値を指定します。メソッドの return ではありません。
  • -> "B" のように単一式を書く場合は yield 不要です。
  • -> { ... } のようにブロックを書く場合、値を返すには yield 値; が必要です。
  • switch式は全分岐で値が決まる必要があります。defaultの有無を確認します。

ループとbreak/continue

for (int i = 0; i < 5; i++) {
  if (i == 1) continue;
  if (i == 3) break;
  System.out.println(i);
}
// 0, 2

continueは次の繰り返しへ、breakはループを抜けます。ネスト時はどのループに効くかを確認します。
