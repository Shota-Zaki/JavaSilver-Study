前置/後置インクリメント

int a = 10;
int b = a++ + a + --a;
// a++ は式には10を渡し、その後aは11
// 次のaは11
// --a は先に10へ減らして式に10を渡す
// b = 31
形式に渡す値変数の変化
++a増やした後の値先に+1
a++増やす前の値後で+1
--a減らした後の値先に-1
a--減らす前の値後で-1

&& と & / || と |

演算子短絡評価意味
&&あり左がfalseなら右は評価しない。
&なしbooleanでは両方評価する。
||あり左がtrueなら右は評価しない。
|なしbooleanでは両方評価する。
int a = 0;
if (false && ++a > 0) {}
System.out.println(a); // 0
if (false & ++a > 0) {}
System.out.println(a); // 1
