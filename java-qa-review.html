質問の論点結論
try内で例外が出たら下の文に行くか対応するcatchへ移動する。try内の残りは実行されない。finallyは原則実行。
interface Worker { void work(); }workを持つ型の契約。実装クラスはpublic void work()を実装する。
抽象メソッド本体なし。具象クラスは実装必須。抽象クラスは未実装のまま持てる。
配列宣言int[] a; / int a[]; はOK。要素数はnew時に指定。
intern()文字列プール上のインスタンス参照を返す。==判定に影響。
ラベルbreak/continueの対象を指定。continueはループラベルに対して使う。
