質問整理まとめ

これまで質問した論点を、試験で使える知識として整理します。

よく質問した論点のまとめ

interface Worker { void work(); } は何を意味するか

Worker型として扱われるものは、work()というメソッドを持つ必要がある、という契約です。実装クラスはpublicでwork()を実装します。

implements と extends の違い

extendsは親から受け継ぐ、implementsはinterfaceのルールを満たす、という違いです。クラスがクラスを継承するならextends、クラスがinterfaceを実装するならimplementsです。

コンパイルエラーと実行時例外の違い

コンパイルエラーは実行前にコードとして不正な状態です。実行時例外はコードとしては成立しているが、実行中の値や状態が原因で発生します。

recordの本質

データをまとめて持つための、変更不可に近いデータ専用クラスです。アクセサはgetName()ではなくname()です。
