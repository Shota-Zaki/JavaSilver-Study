Java Silver Reference

単語集

頻出語を、短い意味と試験での注意に絞って整理します。

JDK

Java開発キット。javacなどを含む。

JRE/JVM

Javaを実行する仕組み。JVMがバイトコードを実行する。

package

クラスの名前空間。ソース先頭に書く。

import

別パッケージの型名を短く使う宣言。

primitive

値そのものを持つ型。int, booleanなど。

参照型

オブジェクトへの参照を持つ型。nullになりうる。

リテラル

コードに直接書く値。10, "abc", trueなど。

var

ローカル変数の型推論。型名ではない。

同一性

同じ実体・同じ参照か。==で判定。

同値性

内容として等しいか。equalsで判定。

immutable

不変。Stringは変更されず新しいStringを返す。

mutable

可変。StringBuilderやArrayListは状態を変える。

static

クラス所属。インスタンスごとではない。

final

再代入不可、継承不可、override不可など対象で意味が変わる。

abstract

未完成。インスタンス化不可、抽象メソッドを持てる。

sealed

継承・実装できる型を制限する。

overload

同名メソッドを引数違いで複数定義。

override

親のメソッドを子で再定義。

polymorphism

親型やinterface型で子の実体を扱う。

cast

参照型を別の型として扱う明示変換。

checked exception

コンパイル時に処理が要求される例外。

unchecked exception

RuntimeException配下など。処理必須ではない。

try-with-resources

AutoCloseableを自動closeするtry。

record

データ保持用の簡潔なクラス記法。
