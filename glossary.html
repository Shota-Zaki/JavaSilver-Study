Java Silver Reference

Java Silver単語集

頻出用語を、意味と試験での注意点に分けて整理します。

JDK / JRE / JVM

意味：JDKは開発用キット、JREは実行環境、JVMはバイトコードを実行する仮想マシン。

試験での注意：javacはJDKに含まれる。javaコマンドはclassファイルをJVM上で実行する。

ソースファイル / クラスファイル

意味：javaファイルをjavacでコンパイルするとclassファイルができる。

試験での注意：publicクラス名とファイル名の一致が必要。

package

意味：クラスの名前空間。分類とアクセス制御に関係する。

試験での注意：package宣言はコメントを除いて先頭。importより前。

import

意味：別パッケージの型名を短く書くための宣言。

試験での注意：ワイルドカードは直下の型だけ。サブパッケージは含まない。

primitive型

意味：値そのものを保持する型。boolean, char, byte, short, int, long, float, double。

試験での注意：nullは代入できない。算術演算ではbyte/short/charがintへ昇格しやすい。

参照型

意味：オブジェクトへの参照を保持する型。String、配列、クラス、インタフェースなど。

試験での注意：変数代入は参照のコピー。オブジェクト本体のコピーではない。

リテラル

意味：コードに直接書いた値。10, 10L, 3.14, "abc" など。

試験での注意：整数は基本int、小数は基本double。floatにはFが必要。

var

意味：ローカル変数の型推論。

試験での注意：フィールド、引数、戻り値型には使えない。nullだけでは型推論不可。

String

意味：不変の文字列クラス。

試験での注意：変更系メソッドは新しいStringを返す。戻り値を受け取らないと変数は変わらない。

StringBuilder

意味：可変の文字列操作クラス。

試験での注意：appendなどは自身を変更する。equalsはStringのような内容比較ではない。

配列

意味：固定長の参照型。lengthフィールドで要素数を取得する。

試験での注意：添字は0始まり。array[array.length]は範囲外。

ArrayList

意味：可変長リスト。add/set/remove/get/sizeが頻出。

試験での注意：setは既存要素の置換。範囲外位置には使えない。

static

意味：クラスに属するメンバ。

試験での注意：staticメソッドからthisや非staticメンバへ直接アクセス不可。

final

意味：再代入不可、オーバーライド不可、継承不可など文脈で意味が変わる。

試験での注意：参照型finalは参照の再代入不可であり、オブジェクト内部の変更不可とは限らない。

this / super

意味：thisは現在のインスタンス、superは親クラス部分を指す。

試験での注意：this(...)やsuper(...)はコンストラクタ先頭にしか書けない。

オーバーロード

意味：同じ名前で引数リストが違うメソッドを定義する。

試験での注意：戻り値だけ違ってもオーバーロードにならない。

オーバーライド

意味：親のメソッドを子で再定義する。

試験での注意：アクセス修飾子は弱められない。戻り値は同じか共変戻り値。

ポリモーフィズム

意味：親型変数で子インスタンスを扱い、実行時の実体に応じてメソッドが動く仕組み。

試験での注意：フィールドは多態的に切り替わらない。メソッドと区別。

interface

意味：実装クラスに振る舞いを要求する型。

試験での注意：抽象メソッドは暗黙にpublic abstract。実装側もpublicが必要。

sealed / non-sealed

意味：継承・実装できる相手を制限または開放する修飾子。

試験での注意：sealedを継承したクラスはfinal/sealed/non-sealedのいずれかが必要。

チェック例外

意味：Exception系のうちRuntimeException以外の多く。

試験での注意：catchまたはthrowsが必要。未処理ならコンパイルエラー。

非チェック例外

意味：RuntimeExceptionとそのサブクラス。

試験での注意：コンパイル時の処理強制はないが、実行時に発生し得る。

try-with-resources

意味：AutoCloseableなリソースを自動でcloseするtry。

試験での注意：closeは宣言と逆順。finallyがあればclose後に実行。
