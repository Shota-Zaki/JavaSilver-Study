Java Silver Reference

単語集

Java Silverで頻出する用語の短い定義。

JDK

Java開発キット。javacなど開発に必要なものを含む。

JRE

実行環境。現在はJDK配布形態との関係で単独意識は薄い。

JVM

classファイルを実行する仮想マシン。

コンパイルエラー

javac段階で失敗。実行されない。

実行時例外

コンパイル後、実行中に発生。

primitive

値そのものを保持する型。

参照型

オブジェクトへの参照を保持する型。

ローカル変数

メソッドやブロック内の変数。自動初期化なし。

フィールド

クラスに属する変数。インスタンス/ static がある。

オーバーロード

同名メソッドを引数違いで複数定義。

オーバーライド

継承先で同じシグネチャのメソッドを再定義。

ポリモーフィズム

宣言型は親でも実体型のオーバーライドメソッドが動く性質。

抽象メソッド

本体を持たず、実装をサブクラスに要求するメソッド。

interface

扱い方の契約。実装クラスは抽象メソッドを実装する。

sealed

継承/実装できる相手を限定する仕組み。

record

値を持つデータ用クラスを簡潔に作る構文。
