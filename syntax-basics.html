Java実行・package/import

javac/java、main、package、importの基本を確認します。

コンパイルと実行

javac Sample.java
java Sample one two

javacはソースをコンパイルし、javaはクラスを実行します。javaコマンドでは拡張子.classを書きません。

mainメソッド

public static void main(String[] args) {}
public static void main(String... args) {}

どちらも起動用mainとして有効です。String args[] も配列宣言としては可能です。

publicクラスとファイル名

// Sample.java
public class Sample {}
class Helper {}

1つのソースファイルにpublicクラスは最大1つ。publicクラス名とファイル名は一致が必要です。

package/importの順序

package com.example;

import java.util.List;

public class Sample {}

package宣言は先頭、importはその後、class宣言はさらに後です。

java.lang

String、System、Integer など java.lang の型はimport不要です。
