Arrays / List

配列・ArrayList・List

配列は固定長、ArrayListは可変長。API、初期値、参照代入の違いが問われます。

配列の基本

int[] a = new int[3];      // {0,0,0}
String[] s = new String[2]; // {null,null}
int[][] m = {{1,2}, {3,4}};
System.out.println(a.length);
項目配列
長さ生成後に固定。
lengthフィールド。length()ではない。
初期値要素は型ごとの初期値。
範囲外ArrayIndexOutOfBoundsException。
型配列自体は参照型。

多次元配列と拡張for

int[][] data = {{1, 2}, {3, 4}};
for (int[] row : data) {
  for (int value : row) {
    System.out.println(value);
  }
}
  • int[][]から1つ取り出すとint[]。
  • int[]から1つ取り出すとint。
  • ジャグ配列では行ごとに長さが違ってよい。

ArrayList

操作例注意
追加list.add("A")末尾追加。
挿入list.add(0,"A")指定位置以降が右へずれる。index範囲に注意。
置換list.set(0,"B")既存要素が必要。空リストには不可。
取得list.get(0)範囲外はIndexOutOfBoundsException。
削除list.remove(0)index指定。
削除list.remove("A")最初に一致した要素を削除。
要素数list.size()配列のlengthと混同しない。

List.of と Arrays.asList

作成方法変更null注意
List.of("A","B")構造変更不可不可add/remove/set不可。
Arrays.asList(array)サイズ変更不可可の場合ありsetは可、add/remove不可。元配列と連動。
new ArrayList<>()可変可通常の可変リスト。
