配列・ArrayList・List

配列は固定長、ArrayListは可変長。宣言・生成・アクセス方法が違います。

配列宣言と生成

int[] a;      // OK
int b[];      // OK
// int[3] c;  // NG: 宣言時に要素数は書かない
int[] d = new int[3];
int[] e = {1, 2, 3};

多次元配列

int[][] a = new int[2][];
a[0] = new int[3];
a[1] = new int[]{10, 20};

2次元配列は「配列の配列」です。内側の長さは揃っていなくてもよいです。

ArrayList

java.util.ArrayList<String> list = new java.util.ArrayList<>();
list.add("A");
list.add("B");
String old = list.set(0, "X"); // oldは"A"
String removed = list.remove(1); // removedは"B"

List.of

List.of(...) は変更不可のListを返します。add や set は実行時例外になります。null要素も不可です。

java.util.List<String> list = java.util.List.of("A", "B");
// list.add("C"); // UnsupportedOperationException
