配列・ArrayList・List

固定長配列、可変長リスト、変更不可リストの違いを整理します。

配列の基本

int[] a = new int[3];
a[0] = 10;
a[1] = 20;
a[2] = 30;
System.out.println(a.length); // 3
// System.out.println(a[3]);  // 実行時例外

配列は固定長です。lengthはメソッドではなくフィールドです。

多次元配列

int[][] a = {{1, 2}, {3, 4, 5}};
for (int[] row : a) {
  for (int v : row) {
    System.out.println(v);
  }
}

int[][]から1つ取り出すとint[]です。行ごとの長さが違うジャグ配列も作れます。

ArrayListの基本操作

ArrayList<String> list = new ArrayList<>();
list.add("A");
list.add("C");
list.add(1, "B"); // A, B, C
list.set(0, "X"); // X, B, C
list.remove("B"); // X, C

add(index, value)は挿入、set(index, value)は置換です。

removeの罠

ArrayList<Integer> list = new ArrayList<>();
list.add(1);
list.add(2);
list.add(3);
list.remove(1); // 添字1を削除 → [1, 3]
list.remove(Integer.valueOf(3)); // 値3を削除 → [1]

int引数なら添字削除、Object引数なら要素削除です。

List.of

List<String> list = List.of("A", "B");
// list.add("C"); // UnsupportedOperationException
// list.set(0, "X"); // UnsupportedOperationException

List.ofの戻り値は変更不可です。null要素も不可です。
