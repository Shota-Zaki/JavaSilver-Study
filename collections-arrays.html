配列・ArrayList・List

配列、ArrayList、List.of、拡張for、length/size、境界例外をまとめます。

配列の宣言と生成

配列変数の宣言では、要素数を書きません。要素数は配列インスタンスを作るときに指定します。

int[] a;      // OK
int b[];      // OK
// int[3] c;  // NG
// int d[3];  // NG

int[] nums = new int[3];
int[] values = {1, 2, 3};

配列はオブジェクトです。参照型なので、変数には配列インスタンスへの参照が入ります。

int[] a = {1, 2, 3};
int[] b = a;
b[0] = 99;
System.out.println(a[0]); // 99

length と範囲

配列の長さはフィールドなので <code>length</code> です。メソッドではありません。添字は0から始まり、最後は <code>length - 1</code> です。

int[] a = {10, 20, 30};
System.out.println(a.length); // 3
System.out.println(a[0]);     // 10
System.out.println(a[2]);     // 30
// System.out.println(a[3]);  // ArrayIndexOutOfBoundsException

ArrayList

ArrayListは可変長のリストです。配列と違って要素数を増減できます。<code>add</code> は追加、<code>set</code> は既存位置の置換です。

ArrayList<String> list = new ArrayList<>();
list.add("A");
list.add("B");
list.set(0, "X");

System.out.println(list); // [X, B]
System.out.println(list.size()); // 2

add(E e)

末尾に追加し、booleanを返します。

add(int index, E e)

指定位置に挿入します。既存要素は後ろへずれます。

set(int index, E e)

既存位置を置き換え、置き換え前の要素を返します。

remove(int index)

指定位置の要素を削除し、削除した要素を返します。

remove(Object o)

一致する要素を削除し、削除できたかbooleanで返します。
intのリストで remove(1) と書くと、値1の削除ではなくインデックス1の削除として解釈されやすい点に注意します。

List.of

<code>List.of</code> は変更不可のListを返します。addやsetを呼ぶとコンパイルは通りますが、実行時にUnsupportedOperationExceptionです。

List<String> list = List.of("A", "B");
System.out.println(list.get(0));
// list.add("C"); // 実行時例外
// list.set(0, "X"); // 実行時例外

拡張for文

拡張for文の変数は、各要素の値または参照を受け取る一時変数です。primitiveやStringのように代入しても、元の配列要素は変わりません。

int[] nums = {1, 2, 3};
for (int n : nums) {
    n = 10;
}
System.out.println(nums[0]); // 1

参照型の場合、一時変数が参照しているオブジェクトの中身を変更すれば、その変更は見えます。ただし、一時変数自体に別オブジェクトを代入しても元の格納先は変わりません。
