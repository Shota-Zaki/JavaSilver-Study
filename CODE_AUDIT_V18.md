# CODE_AUDIT_V18

## 対象

起点: `JavaSilver-Study-indent-fix-v17.zip`

## 目的

第2章 問23 のインデント崩れ修正後、同種のコード再現ミス・答えとの整合崩れが他に残っていないかを確認する。

## 修正した問題

### ch02-q22 テキストブロックの構文

PDF問題画像と解説画像を確認した結果、v17には次の不整合があった。

- 選択肢Aが三重引用符のテキストブロックとして登録されていたが、PDF上は通常のダブルクォーテーションを使った複数行文字列で、コンパイルエラーになる選択肢。
- 選択肢Dが通常のダブルクォーテーション開始として登録されていたが、PDF上は三重引用符で開始し、開始直後に改行がある正しいテキストブロック。
- 正解キーがAになっていたが、PDF解説ではD。

対応:

- `ch02-q22.options[A].code` を通常文字列の複数行コードに修正。
- `ch02-q22.options[D].code` をPDF上の三重引用符テキストブロックに修正。
- `ch02-q22.answer` を `D` に修正。
- 解説・選択肢別解説・関連知識・試験注意点・解き方手順を修正。
