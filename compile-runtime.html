コンパイルエラーと実行時例外

失敗が「実行前」か「実行中」かを分けます。

失敗段階の違い

段階意味例
コンパイルエラーJavaとして実行可能なclassを作れない型不一致、未初期化、到達不能、チェック例外未処理。
実行時例外classは作れるが、実行中の値で失敗NPE、配列範囲外、ClassCastException、NumberFormatException。
正常出力コンパイルも実行も通る副作用と参照先を最後まで追う。

コンパイルエラー例

int x;
// System.out.println(x); // 未初期化

byte b = 10;
// b = b + 1; // intをbyteへ代入不可

// return; System.out.println("x"); // 到達不能

実行時例外例

String s = null;
// s.length();

int[] a = new int[1];
// a[1] = 10;

Integer.parseInt("abc");
