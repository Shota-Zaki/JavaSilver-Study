頻出メソッド

String、StringBuilder、ArrayList、ラッパークラスの頻出メソッドを、戻り値と副作用に注目して整理します。

String

Stringメソッドは元の文字列を変更しません。結果を使うなら戻り値を受け取ります。

length()

文字数を返します。配列の length とは違いメソッドです。

charAt(int)

指定位置のcharを返します。範囲外なら例外。

substring(begin, end)

beginを含み、endを含まない部分文字列を返します。

equals / equalsIgnoreCase

内容比較。呼び出し元がnullならNPE。

indexOf

見つからなければ -1。

replace

置換後の新しいStringを返します。

StringBuilder

StringBuilderは可変です。多くの操作が自分自身を返します。

append

末尾追加。戻り値は同じStringBuilder。

insert

指定位置に挿入。位置が範囲外なら例外。

delete(begin,end)

beginを含み、endを含まない範囲を削除。

reverse

自身を反転します。

ArrayList / List

size()

要素数。配列の length とは違います。

get(index)

指定位置の要素を返します。範囲外なら例外。

set(index, value)

指定位置を置換し、古い要素を返します。サイズは変わりません。

add(value)

末尾に追加します。

remove(index/object)

オーバーロードに注意。remove(1) はindex指定です。

contains

equals による同値判定です。

Integer / ラッパークラス

parseInt(String)

文字列をintへ変換。変換不能なら NumberFormatException。

valueOf

ラッパー型インスタンスを返します。

xxxValue

intValue() などでプリミティブ値を取り出します。

オートボクシング

プリミティブとラッパーの自動変換。ただしnullのアンボクシングはNPE。
