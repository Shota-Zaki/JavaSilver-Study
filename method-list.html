頻出メソッド一覧

戻り値・副作用・例外の有無を中心に、Java Silverで間違えやすいメソッドをまとめます。

String

メソッド戻り値副作用注意
length()intなし配列は length フィールド
charAt(int)charなし範囲外で例外
substring(int,int)Stringなし終了位置は含まない
replaceStringなし元のStringは変わらない
concatStringなし戻り値を受け取る
internStringなし文字列プールの参照を返す

StringBuilder

メソッド戻り値副作用注意
appendStringBuilderあり同じインスタンスを返す
insertStringBuilderあり位置指定に注意
deleteStringBuilderありendは含まない
reverseStringBuilderあり元が逆順になる
substringStringなしBuilder本体は変わらない

ArrayList / List

メソッド戻り値副作用注意
size()intなし配列は length
add(E)booleanあり末尾追加
add(int,E)voidあり挿入
remove(int)削除した要素ありインデックス指定
remove(Object)booleanあり値指定
set(int,E)置換前の要素あり要素数は変わらない

Wrapper / parse

メソッド戻り値注意
Integer.parseInt(String)int変換不可なら NumberFormatException
Integer.valueOf(String)Integer戻り値はラッパー
String.valueOf(Object)Stringnullなら文字列 "null"
Objects.equals(a,b)booleannull安全な比較

覚える基準

丸暗記ではなく、次の基準で分類してください。

  • 戻り値を受け取らないと意味がない: String 系。
  • 本体を変える: StringBuilder の編集系、ArrayList の追加削除。
  • 境界で例外: charAt、substring、get。
  • 見つからなくても例外ではない: indexOf は -1。
