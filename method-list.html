頻出メソッド・境界値

戻り値、変更有無、例外をメソッドごとに確認。

String

メソッド戻り値例外/注意
length()int配列はlengthフィールド。
charAt(int)char範囲外で例外。
substring(begin,end)Stringendは含まない。元Stringは変わらない。
indexOf(String)intなければ-1。
replaceString戻り値を使わないと変更なし。
equalsbooleannull左辺に注意。

StringBuilder

メソッド戻り値注意
appendStringBuilder自分を変更。
insertStringBuilder範囲に注意。
deleteStringBuilderendは含まない。
reverseStringBuilder自分を変更。
toStringStringStringへ変換。

List / ArrayList

メソッド戻り値注意
add(E)boolean末尾追加。
add(index,E)void挿入。
set置換前の要素追加ではない。
remove(index)削除した要素Integerリストで混同注意。
size()int配列はlength。

ラッパー / Object / System

項目戻り値注意
Integer.parseIntint変換不可でNumberFormatException。
Integer.valueOfIntegerラッパーを返す。
Object.toStringStringnullに対して呼べない。
System.out.printlnvoid代入できない。
