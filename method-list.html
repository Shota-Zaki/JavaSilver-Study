頻出メソッド

String、StringBuilder、ArrayList、List、Object、Integerの戻り値・副作用・境界例外をまとめます。

String

length()

文字数をintで返します。配列のlengthとは違いメソッドです。

charAt(index)

指定位置のcharを返します。範囲外はStringIndexOutOfBoundsException。

substring(begin, end)

begin以上end未満の部分文字列を返します。元のStringは変わりません。

indexOf(str)

見つかれば開始位置、見つからなければ-1。

equals(obj)

内容比較。nullに対して呼ぶとNPE。

equalsIgnoreCase(str)

大文字小文字を無視して比較。呼び出し元がnullならNPE。

StringBuilder

append(value)

自分自身を変更し、自分自身を返します。

insert(index, value)

指定位置に挿入します。

delete(begin, end)

begin以上end未満を削除します。

replace(begin, end, str)

範囲を置き換えます。

reverse()

文字順を反転します。

toString()

Stringへ変換します。

ArrayList / List

size()

要素数を返します。

get(index)

指定位置の要素を返します。

add(e)

末尾に追加し、booleanを返します。

add(index, e)

指定位置に挿入します。

set(index, e)

指定位置を置換し、古い要素を返します。

remove(index)

指定位置を削除し、削除した要素を返します。

remove(object)

一致する要素を削除し、booleanを返します。

Object / Integer / System

toString()

文字列表現を返します。recordでは自動生成されます。

equals(Object)

Objectの標準実装は参照比較。Stringやrecordは内容比較へオーバーライド。

hashCode()

equalsと整合するハッシュ値。recordでは自動生成。

Integer.parseInt(String)

文字列をintへ変換。変換不能ならNumberFormatException。

System.out.println(...)

表示して改行します。戻り値はvoidなので代入不可。
