メソッド・コンストラクタ

戻り値、引数、オーバーロード、this/super、コンストラクタの制約を整理します。

メソッド宣言

要素例注意
戻り値int calc()戻り値型に合う値をreturnする。voidは値を返さない。
引数test(int a, String b)型と順序がシグネチャに含まれる。
可変長引数test(String... s)最後の引数だけ。配列として扱える。
returnreturn value;voidならreturn;だけ可能。

オーバーロード

void test(int a) {}
void test(long a) {}
void test(Integer a) {}
void test(int... a) {}

test(10); // 基本は完全一致 int が優先

オーバーロードはコンパイル時に、引数の型で決まります。戻り値型だけ違ってもオーバーロードにはなりません。

コンストラクタ

class Sample {
    Sample() { }
    void Sample() { } // コンストラクタではなく通常メソッド
}
  • クラス名と同じで戻り値型なし。
  • 戻り値型を書くと通常メソッド。
  • 自分でコンストラクタを書くと、デフォルトコンストラクタは自動生成されない。
  • this() / super() はコンストラクタの先頭文だけ。
