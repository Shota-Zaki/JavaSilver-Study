メソッド・コンストラクタ

宣言、呼び出し、オーバーロード、初期化順を整理します。

メソッド宣言の基本

public static int add(int a, int b) {
  return a + b;
}

戻り値型、メソッド名、引数リスト、returnの整合性を確認します。void以外は値を返す必要があります。

オーバーロード

void test(int a) {}
void test(long a) {}
void test(Integer a) {}
void test(int... a) {}

引数リストが違えばオーバーロードです。呼び出し時は完全一致、拡大変換、ボクシング、可変長の優先度を意識します。

コンストラクタ

class Sample {
  Sample() {
    this("default");
  }
  Sample(String name) {
    System.out.println(name);
  }
}

コンストラクタに戻り値型はありません。this(...)やsuper(...)は先頭行だけです。

super()の暗黙挿入

class Parent {
  Parent(String name) {}
}
class Child extends Parent {
  // Child() {} は暗黙のsuper()が入り、親に引数なしがないのでエラー
  Child() { super("x"); }
}

親に引数なしコンストラクタがない場合、子は明示的にsuper(引数)を呼ぶ必要があります。
