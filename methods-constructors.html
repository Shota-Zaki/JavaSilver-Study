コンストラクタ

ルール注意
戻り値型を書かないvoidを書くと普通のメソッドになる。
クラス名と同じ大文字小文字も一致。
this(...)同じクラスの別コンストラクタ呼び出し。先頭のみ。
super(...)親コンストラクタ呼び出し。先頭のみ。
省略時super()が暗黙挿入される。親に無引数がないと注意。
継承コンストラクタは継承されない。

staticメソッド

class Sample {
  int value = 10;
  static int count = 0;
  static void test() {
    // System.out.println(value); // 不可
    System.out.println(count);    // 可
  }
}
  • staticから非staticへ直接アクセス不可。
  • インスタンス経由でstaticにアクセスできても推奨ではなく、試験では混乱材料。
