正解：D

テキストブロックについての問題です。
テキストブロックが開始されるトリプルクォーテーションの後は改行しなければいけません。記述できるのは改行やスペース、タブ文字といった空白文字です。たとえば、次のように最初のトリプルクォーテーションの後に文字列を記述するとコンパイルエラーとなります。

例: トリプルクォーテーションの後に改行がない（コンパイルエラー）

```java
String msg = """this is
sample text
""";
```

一方で、テキストブロックの終わりを示すトリプルクォーテーションの前であれば、次の例のように自由に文字列リテラルを記述できます。

例: テキストブロックの末尾は改行は不要

```java
String msg = """
this is sample text""";
```

選択肢AとBは、トリプルクォーテーションではなくダブルクォーテーションを使っています。また、選択肢CとEは、はじまりのトリプルクォーテーションの後で改行せずに文字列リテラルを記述しています。そのため、これらの選択肢は誤りです。テキストブロックは、はじまりのトリプルクォーテーションの後で改行を入れなければいけません。以上のことから、選択肢Dが正解です。
