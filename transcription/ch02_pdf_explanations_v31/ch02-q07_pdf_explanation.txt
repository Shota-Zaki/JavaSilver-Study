正解：C

varを使った変数宣言に関する問題です。
varによる型推論は、ローカル変数の宣言にしか使えません。
フィールドの宣言にはvarを使って型推論を行うことができないため、次のよ
うなコードはコンパイルエラーとなります。

例: フィールドの宣言に型推論は使えない（コンパイルエラー）

```java
public class Sample {
    var name = "";
    var price = 0;
}
```

また、次のような引数宣言のコードもコンパイルエラーとなります。varは値
の代入式を見て、代入する値の型から変数の型を推論します。メソッドの定
義をコンパイルしようとしたときに、どのコードからこのメソッドが使われ
ているかを探して型推論することはできないため、コンパイルエラーになり
ます。

例: 引数の型宣言に型推論は使えない（コンパイルエラー）

```java
public class Sample {
    public void test(var value) {
        System.out.println(value);
    }
}
```

設問のコードでは、Sampleクラスのフィールドの宣言にvarを使っています。
また、コンストラクタの引数宣言にもvarを使っているため、コンパイルエ
ラーとなります。よって、選択肢Cが正解です。
