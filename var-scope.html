var・スコープ・初期化

varは型名ではなく、ローカル変数の型推論です。

varが使える場所

var name = "java";      // String
var list = new java.util.ArrayList<String>();
for (var s : list) { }
try (var r = new java.io.StringReader("a")) { }

varが使えない場所

場所例理由
フィールドprivate var value;フィールドには使えない。
メソッド引数void test(var x)引数には使えない。
戻り値型var test()戻り値型には使えない。
初期値なしvar x;推論材料がない。
nullのみvar x = null;型を決められない。
配列初期化子のみvar a = {1,2};配列型を推論できない。
